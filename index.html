How to make it responsive...

HTML tables allow web developers to arrange data into rows and columns.

You cannot style a traditional HTML table to be Responsive in the modern sense.

The simplest way of making a table responsive is to give it an overflow-x auto value.

A responsive table will display a horizontal scroll bar if the screen is too small
